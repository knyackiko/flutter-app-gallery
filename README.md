## GitHubリポジトリ検索アプリ (github_repo_app)

- リポジトリ/ユーザ検索

  - Shimmerの使用
- リポジトリ詳細表示

- お気に入り登録
  - ローカルDB(SQLite)使用
- その他

  - ユニットテスト
